# Power Apps PCF control to display text with drop shadow

![Drop Shadow Text Control](Demos/../../Demos/Drop&#32;shadow&#32;text&#32;in&#32;Model-Drive&#32;Apps_small.gif)

Following properties are configurable

- Text
- Color
- Font
- Font size
- Font weight
- Shadow color
- Shadow blur
- Shadow size
- Shadow x offset
- Shadow y offset
